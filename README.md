Calculator 🧮
A simple calculator built using Python and Streamlit.

Features

- Addition (+)
- Subtraction (-)
- Multiplication (×)
- Division (÷)
- Clear Button (C)
- User-friendly interface
- Responsive design

Technologies Used

- Python
- Streamlit

How to Run

1. Install Streamlit:

pip install streamlit

2. Run the project:

streamlit run project1.py
